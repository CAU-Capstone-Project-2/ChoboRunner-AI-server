\# ChoboRunner AI Server



러닝 자세 실시간 분석 서비스 — AI 서버



캡스톤 디자인(2) 프로젝트 / 중앙대학교 소프트웨학부



\## 개요



측면 단일 카메라로 촬영한 러닝 영상에서 자세 지표(Trunk Lean, Initial Knee Flexion, Foot Strike Pattern)를 실시간으로 분석하고 초보 러너에게 참고용 피드백을 제공합니다.



> ⚠️ 본 서비스는 의료 진단 목적이 아닌 참고용 피드백을 제공합니다.



\## 기술 스택



\- Python 3.11

\- MediaPipe Tasks (Pose Landmarker)

\- OpenCV, NumPy

\- FastAPI + Pydantic

\- pytest
